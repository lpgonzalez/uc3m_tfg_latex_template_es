\section{Listas simples}

Las listas son sencillas. Para crearlas:
\begin{itemize}
  \item Las entradas de la lista comienzan con el comando \verb|\item|.
  \item En este tipo de lista cada entrada empieza con un ``bullet''.
  \item La longitud de cada entrada es arbitraria.
  \item Y el número de elementos también lo es.
\end{itemize}

% -------------------------

\section{Listas personalizadas}

Tanto las listas con puntos como las enumeradas permiten cambiar el elemento
inicial con \verb|\item[elemento]|:
\begin{enumerate}
  \item Primer punto.
  \item Segundo punto.
  \item[!] Tercer punto, comenzando con una exclamación.
  \item[$\blacksquare$] Ahora empezando con un cuadrado negro.
  \item[NOTA] Esta entrada no tiene ``bullet''.
  \item[] ¿Entrada en blanco?
\end{enumerate}

\section{Ecuaciones}

% El paquete amsmath se carga en tfg_uc3m.sty.
\begin{equation}
    \label{eq:euler}
    e^{i\pi} + 1 = 0
\end{equation}

La ecuación \ref{eq:euler} es la identidad de Euler. También se pueden escribir
fórmulas en línea, por ejemplo $a^2 + b^2 = c^2$.

% Varias ecuaciones alineadas por el signo "=" con el entorno align (amsmath).
% Usa align* (con asterisco) si no quieres numerarlas.
\begin{align}
  (a+b)^2 &= a^2 + 2ab + b^2 \\
  (a-b)^2 &= a^2 - 2ab + b^2
\end{align}

\section{Acrónimos y glosario}

El \gls{GULa} promueve el \gls{swlibre} en la \gls{UC3Ma}. Por ejemplo,
GNU/Linux es un \gls{SOa} libre.\glsadd{GULg}\glsadd{UC3Mg} En la primera
aparición se muestran la forma desarrollada y la sigla; después, los acrónimos
aparecen abreviados. Las siglas se recogen en el apartado de acrónimos y los
